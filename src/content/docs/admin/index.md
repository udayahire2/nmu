## Dashboard

The admin dashboard at `/admin/dashboard` gives a bird's-eye view of platform activity:

- Total students enrolled
- Pending content approvals
- Recent uploads and activity
- Active faculty members

## Admin Navigation

| Route | Description |
|-------|-------------|
| `/admin/dashboard` | Overview statistics |
| `/admin/approvals` | Review pending resources |
| `/admin/students` | Manage student accounts |
| `/admin/faculty` | Manage faculty accounts |
| `/admin/syllabus` | Edit curriculum structure |
| `/admin/resources` | Browse all platform resources |
| `/admin/settings` | Platform-wide settings |

## Quick Actions

From the dashboard you can:

- **Approve** or **Reject** the latest pending submissions
- **Suspend** user accounts
- **Export** user or resource data as CSV
- View platform-wide analytics at a glance
