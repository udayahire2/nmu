## Overview

Welcome to **UDX** — a centralized academic platform built for students, faculty, and administrators. UDX makes it easy to share, discover, and manage educational resources organized by branch, semester, and subject.

Whether you're a student looking for lecture notes or an admin managing platform content, this documentation covers everything you need.

## Key Features

- 📚 **Study Materials** — Notes, PDFs, and video lectures organized by branch → semester → subject.
- 🗂️ **Syllabus Management** — Structured curriculum trees with subject codes and topic breakdowns.
- 👥 **Role-Based Access** — Dedicated dashboards for students, faculty, and administrators.
- 🔍 **Smart Search** — Full-text search across all documentation and resources.
- 🌙 **Dark Mode** — Beautiful dark and light themes out of the box.

## Platform Roles

| Role | Description |
|------|-------------|
| **Student** | Browses and downloads study materials |
| **Faculty** | Uploads resources, manages subject content |
| **Admin** | Approves content, manages users and settings |

## Prerequisites

Before you get started, make sure you have:

- Node.js `18+` installed
- A Supabase account and project configured
- Basic knowledge of React and TypeScript
