## Structure

Study materials follow a strict four-level hierarchy:

```
Branch → Semester → Subject → Topics → Resources
```

Each **topic** can have multiple attached resources (PDFs, videos, and notes).

## Browsing Materials

Students navigate to **Resources** from the top nav:

1. Select your **Branch** (e.g. Computer Engg.)
2. Select your **Semester** (1–8)
3. Browse subjects and their topics
4. Click a resource to view or download it

## Resource States

| State | Description |
|-------|-------------|
| `pending` | Uploaded by faculty, awaiting admin approval |
| `approved` | Visible to all students |
| `rejected` | Flagged by admin, not visible |

## Search

Use the search bar at the top to find resources by title, subject, or topic across all branches and semesters.
