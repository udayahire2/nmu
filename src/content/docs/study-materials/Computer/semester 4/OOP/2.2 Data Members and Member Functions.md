### **Data Members**

Data members are **variables declared inside a class**. They represent the **properties or state** of an object. Each object of a class gets its **own separate copy** of data members. Data members can be declared as `private`, `protected`, or `public` to control access.

### **Member Functions**

Member functions are **functions defined inside a class**. They define the **behavior or operations** that can be performed on the data members. Member functions can access private data members directly and are used to manipulate object data.

---

### **Key Characteristics**

- Data members store information.
    
- Member functions operate on that information.
    
- Both are defined inside a class.
    
- Together they implement **encapsulation**.
    

---

### **Types of Data Members**

1. **Instance data members** – Separate copy for each object.
    
2. **Static data members** – Shared among all objects of the class.
    

---

### **Types of Member Functions**

1. **Instance member functions** – Work on object data.
    
2. **Static member functions** – Work with static data members only.
    

---

### **Difference Between Data Members and Member Functions**

|**Data Members**|**Member Functions**|
|---|---|
|Variables of a class|Functions of a class|
|Store object data|Perform operations on data|
|Represent state|Represent behavior|
|Occupy memory per object|Code shared among objects|
