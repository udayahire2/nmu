## Upload Workflow

Faculty members upload content via the **Faculty Dashboard** at `/dashboard/faculty`.

1. Navigate to `/dashboard/faculty`
2. Select the target **Subject** from your assigned list
3. Click **Upload Resource**
4. Fill in the resource form:
   - **Title** — Short, descriptive name
   - **Description** — What this resource covers
   - **Type** — Notes / PDF / Video / Link
   - **File** — Upload a file or paste a URL
5. Submit — the resource enters `pending` state
6. Admin reviews and approves (or rejects) the submission
7. Approved resources become visible to all students

## File Limits

| Type | Max Size |
|------|----------|
| PDF | 25 MB |
| Video | 500 MB |
| Notes (text) | 5 MB |
| External Link | — |

## Tips for Good Resources

- Use clear, descriptive titles (e.g. `Unit 3 — Linked Lists Notes`)
- Match the resource to the correct topic so students can find it easily
- Prefer PDF or text notes over raw image scans for accessibility
